**Goal:** End-to-end cloud deployment with integrated security & observability  
**Tech Stack:** AWS EC2, Docker, Jenkins, ArgoCD, Kubernetes, Terraform, Trivy, SonarQube, Grafana  

**Pipeline Flow:**
